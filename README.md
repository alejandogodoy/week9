## Install

- Install Node.js and npm : https://nodejs.org/en/
- Install webdriverIO : npm install @wdio/cli
- Config webdriverIO : npx wdio config (select Jasmine Framework)
- Install wdio/sync : npm install @wdio/sync

## Run test

npm run test
